如何使用

配置

Linux

编辑

/etc/docker/daemon.json
添加

    {
      "allow-nondistributable-artifacts": ["docker.hunantv.com"]
    }

Mac

点击顶部菜单栏docker图标 preference -> Daemon -> Insecure registries里面增加 docker.hunantv.com 

    {
      "allow-nondistributable-artifacts": ["docker.hunantv.com"]
    }

windows

仅支持win10专业版和企业，操作和mac类似

修改完成后，重启docker

登录仓库

请向RTX:胡瀛寰 申请账户

    docker login docker.hunantv.com

推送仓库

先给自己image打上标签

    docker tag yourself-image:version  docker.hunantv.com/用户名/image-name:version
    docker push docker.hunantv.com/用户名/image-name:version

注意目前只可推送自己名下的image， 即打tag时，必须为 

用户名/image-name:tag
后续开放group模式， 公共image，将推送到
lib
目录下，仅管理员具有权限，如若有公共image需求请联系管理员
